 Iris Flower Classification

 Project Overview

This project uses Machine Learning to classify Iris flowers into three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The classification is based on flower measurements such as sepal length, sepal width, petal length, and petal width.

 Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* Scikit-learn
* KaggleHub

 Machine Learning Model

* Random Forest Classifier

 Project Workflow

1. Download and load the Iris dataset.
2. Perform data preprocessing.
3. Visualize the dataset using pair plots.
4. Split the data into training and testing sets.
5. Train a Random Forest Classifier model.
6. Evaluate model performance using accuracy score and classification report.
7. Predict the species of a new flower sample.

 Results

* Successfully classified Iris flower species.
* Achieved high prediction accuracy on the test dataset.
* Generated visualizations for better understanding of the dataset.

Conclusion

This project demonstrates the implementation of a supervised machine learning classification model using the Iris dataset. The Random Forest Classifier provides accurate predictions and helps understand fundamental machine learning concepts.
